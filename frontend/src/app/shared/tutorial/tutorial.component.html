Lets Get Started!


Start by adding a Machine. It's not going to give you any options for the job or op field, but that is perfectly normal.

You will need to have at least 1 Machine and 1 Part to be able to add a Job, and at least 1 Job to assign an Op to.

Now that you have created a machine, lets add your first part! A Part only needs to have a "part number." The same Part can be used on as many jobs as you need it for.
You will need to have at least 1 Part to be able to add a Job

Now that you have created your first part, we can add our first Job. A job holds all of your information for the material recieved for the job.
By inputing information for the first lathe operation, you will be able to calculate a maximum production for the material recieved.

Now that you have created your first job, you will be able to use the production calculator features for that job.
Now lets add an Operation to the job below, which will let us create production lots and hourly production, after we assign a job and op to our machine.

Now that you have created your first job and operation, you will be able to use the production tracking features for that job.
Lets start by using the Hourly Tracking where you will see a button that allows you to asign a job and op to your machine if you have not already done so. Once a job is assigned, you can add a production count by clicking on "quick input."

Now that we have our first hourly production count, lets try adding a Production Lot. Your production lots will be used to track how many parts have been made to an operation on the job so far.

After adding Multiple Production Lots for one job, the Production By Job and Op page will show total parts to date for the operation. This can also be reached by using the dropdown on the Production page, and checking the production for the current job and operation on the selected machine.
